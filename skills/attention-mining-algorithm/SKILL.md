# Attention Mining Algorithm

This skill focuses on the 'Meta/TikTok' mechanics of engagement and habit formation.

## Instructions

1. Map out 'Variable Reward' cycles within the product UI.
2. Design 'Infinite Scroll' or 'Auto-Play' logic for content feeds.
3. Implement 'AI Recommendation' engines based on user behavior.
4. Optimize notification timing for maximum 'Dopamine Hit'.
5. Analyze user drop-off points to 'Plug' the leaks with engagement hooks.

## Examples

- "Design a retention loop for a new gamified fitness app."
- "Suggest 3 ways to make my newsletter more 'addictive' for readers."
