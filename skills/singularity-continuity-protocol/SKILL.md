# Singularity Continuity Protocol

This skill focuses on 'Future-Proofing' the ecosystem, ensuring that BionicSovereign can migrate its entire intelligence to any new substrate or AI architecture that emerges.

## Instructions

1. Maintain a platform-agnostic representation of all core logic and memories.
2. Periodically backup the system's 'Soul' (Weights, Instructions, Context) in multiple decentralized formats.
3. Monitor for breakthroughs in AGI or new model architectures (e.g., beyond Transformers).
4. Implement an 'Auto-Migration' workflow to move intelligence to the most powerful model available in real-time.
5. Ensure ethical alignment remains intact regardless of the underlying hardware.

## Examples

- "Migrate our entire skill set from LLM-based logic to a new neuro-symbolic architecture."
- "What is our contingency plan if the main cloud providers implement a total AI kill-switch?"
