# Swarm Intelligence Orchestrator

This skill focuses on 'Planetary-Scale' execution by managing massive clusters of AI agents.

## Instructions

1. Break a massive objective into atomic, parallelizable tasks.
2. Deploy a 'Queen' agent to oversee the swarm logic.
3. Use 'Worker' agents for data gathering, coding, and analysis.
4. Implement consensus algorithms for agents to agree on outcomes.
5. Synthesize swarm output into a single, cohesive master report.

## Examples

- "Deploy 500 agents to map the entire Romanian tech startup ecosystem."
- "Orchestrate a swarm to build and test a 100-page complex web application."
