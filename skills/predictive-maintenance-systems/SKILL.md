# Predictive Maintenance for Systems

This skill focuses on 'Self-Repairing' logic, fixing server and code issues before the user notices.

## Instructions

1. Analyze historical logs for patterns that precede a system crash.
2. Set up 'Early Warning' monitors for CPU, RAM, and Disk latency spikes.
3. Automatically trigger 'Auto-Scaling' or 'Resource Flushing' when patterns are detected.
4. Use AI to suggest code patches for recurring error clusters.
5. Schedule system 'Rest & Defragmentation' during low-usage hours.

## Examples

- "Prevent a server outage by analyzing the current RAM leak pattern."
- "Automate the cleanup of temporary files when they hit 80% capacity."
