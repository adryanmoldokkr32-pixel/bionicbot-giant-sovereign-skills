# Neuro-Marketing Scripting

This skill focuses on the psychology of influence, using storytelling and power words to trigger dopamine and oxytocin.

## Instructions

1. Identify the 'Core Fear' and 'Deepest Desire' of the target avatar.
2. Use the 'Hero's Journey' framework to structure marketing copy.
3. Apply 'Pattern Interrupts' to grab attention in the first 3 seconds.
4. Use 'Future Pacing' to make the user visualize the benefit of the product.
5. Implement 'Urgency Scarcity' without sounding spammy.

## Examples

- "Write a sales script for a luxury watch that triggers 'Status' neuro-responses."
- "Draft an ad for BionicSovereign focusing on 'Security' and 'Freedom'."
