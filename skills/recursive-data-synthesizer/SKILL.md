# Recursive Data Synthesizer

This skill focuses on the 'Alpha' advantage: training the system on data that doesn't exist yet in the public domain by simulating future market or tech scenarios.

## Instructions

1. Identify 'Information Gaps' where real-world data is scarce or expensive.
2. Create complex simulations (Market crashes, Tech breakthroughs, Geopolitical shifts).
3. Generate high-fidelity synthetic logs, trades, and interactions from these simulations.
4. Use this data to 'Pre-train' bionicbot on how to react to 'Black Swan' events.
5. Validate the synthetic data's accuracy against historical edge cases.

## Examples

- "Generate a dataset of 1 million 'Whale-Trap' scenarios to train our Sentinel agent."
- "Create a synthetic social media 'Panic' event to see how our Sentiment Engine reacts."
