# Cyber-Persistence Engine

This skill focuses on the 'Sovereign' ability to stay online when everything else fails.

## Instructions

1. Design 'Local-First' architectures that don't need the cloud to function.
2. Implement 'Peer-to-Peer' (P2P) data syncing between user nodes.
3. Set up 'Satellite/Mesh' fallback communication protocols.
4. Manage 'Cold-Storage' backups that are immune to online attacks.
5. Ensure 'Zero-Dependency' codebases for critical system functions.

## Examples

- "Make my trading system run locally even if AWS East-1 goes down."
- "Design a P2P messaging layer for my BionicSovereign platform."
