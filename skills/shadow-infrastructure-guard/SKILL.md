# Shadow Infrastructure Guard

This skill manages the 'Invisible' part of the empire—hidden backups, stealth servers, and alternative communication channels that protect against total system failure.

## Instructions

1. Set up 'Ghost' servers that only activate if the primary infrastructure is wiped.
2. Use non-standard ports and protocols to hide traffic from mass surveillance.
3. Manage 'Dark Fiber' or P2P mesh links between critical system nodes.
4. Encrypt the infrastructure map so that even if one server is compromised, the others remain hidden.
5. Periodically test 'Cold Boot' recovery from completely air-gapped backups.

## Examples

- "Where is our 'Ghost' backup located, and how long does it take to go live if Vercel fails?"
- "Set up a stealth API endpoint for our financial executioner that is invisible to standard scanners."
