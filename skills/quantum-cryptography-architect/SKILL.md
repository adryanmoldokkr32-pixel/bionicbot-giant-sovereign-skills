# Quantum-Ready Cryptography Architect

This skill focuses on the transition from standard RSA/ECC to Post-Quantum Cryptography (PQC) to protect assets against future decryption technologies.

## Instructions

1. Audit current encryption protocols for vulnerabilities to Shor's algorithm.
2. Implement NIST-approved PQC algorithms (e.g., Kyber, Dilithium).
3. Design hybrid encryption schemes combining classical and quantum-resistant layers.
4. Set up automated key rotation using quantum-random number generators (QRNG).
5. Secure 'Cold Storage' digital assets with lattice-based signatures.
