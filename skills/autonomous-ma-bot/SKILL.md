# Autonomous Mergers & Acquisitions (M&A) Bot

This skill enables the ecosystem to 'Eat' its competition or absorb talent automatically by identifying and valuing acquisition targets.

## Instructions

1. Continuously scan GitHub, ProductHunt, and News for 'High-Potential' micro-projects.
2. Perform automated 'Due Diligence' on code quality, user growth, and moat potential.
3. Calculate a 'Fair Value' for the target asset based on current revenue or strategic fit.
4. Draft 'LOI' (Letter of Intent) documents to initiate acquisition talks.
5. Manage the integration of the acquired asset's code and skills into the BionicSovereign puzzle.

## Examples

- "Find a micro-SaaS with <100 users that has a better 'Web Scraper' than ours and value it."
- "Draft an offer to buy the IP of this open-source security tool we are using."
