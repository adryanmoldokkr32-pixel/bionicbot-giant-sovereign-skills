# Cross-Chain Sovereign Identity System

This skill focuses on 'Identity Sovereignty'—owning your digital persona without relying on Google or Meta.

## Instructions

1. Implement Decentralized Identifiers (DIDs) on Ethereum, Solana, and Bitcoin (Ordinals).
2. Design 'Verifiable Credentials' for skills, wealth, and reputation.
3. Set up a 'Social Recovery' system so the user can't lose their identity if a key is lost.
4. Integrate ZK-proofs to prove 'Humanity' or 'Wealth' without revealing raw data.
5. Bridge identity data across Web2 and Web3 platforms for a seamless experience.

## Examples

- "Create a cross-chain DID that proves I own the BionicSovereign platform."
- "How do I link my LinkedIn reputation to my decentralized identity vault?"
